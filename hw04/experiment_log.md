# 实验记录
## 1. 环境说明
- 操作系统：Windows 11
- Python版本：3.10
- 运行设备：普通笔记本电脑（无独立GPU）
- 依赖安装命令：`pip install -r requirements.txt`

## 2. 测试说明
- 测试音频：任务二导出的 `voice_demo.mp3`，时长约40秒，为中文科普文本配音。
- 测试方式：运行 `asr_test.py` 对音频文件进行离线识别。

## 3. 识别样例
- 原始文本：当你对着手机说“帮我定明天早上7点的闹钟”，或是刷短视频时自动弹出字幕，背后都是AI语音识别（ASR）在工作。
- 识别结果：当你对着手机说“帮我定明天早上7点的闹钟”，或是刷短视频时自动弹出字幕，背后都是AI语音识别（ASR）在工作。
- 延迟情况：base模型在CPU上识别40秒音频，耗时约8秒，无明显卡顿。

## 4. 实验结论
Whisper对本次测试的中文配音音频识别准确率接近100%，断句和标点处理也比较准确，完全满足本次作业的要求，且部署和运行门槛极低，适合新手快速完成实验。
