# hw04 作业说明
本目录为人工智能科技前沿课程第4次作业，包含以下文件：

| 文件名 | 用途说明 |
| :--- | :--- |
| text_gen.md | 任务一：大模型生成的科普文稿 |
| jianying.md | 任务二：剪映声音克隆/配音说明 |
| voice_demo.mp3 | 任务二导出的配音音频文件 |
| asr_report.md | 任务三：开源ASR方案对比报告 |
| experiment_log.md | 任务三：ASR实验过程与结果记录 |
| requirements.txt | 任务三Python依赖清单 |
| asr_test.py | 任务三：Whisper语音识别可运行代码 |
| asr_result.txt | 任务三：语音识别输出的文本结果 |

## 运行说明
1.  安装依赖：`pip install -r requirements.txt`
2.  运行识别：`python asr_test.py`
