# 开源语音识别（ASR）方案对比报告
## 1. 对比方案列表
本次调研对比了3种主流开源ASR方案：OpenAI Whisper、Vosk、FunASR。

## 2. 多维度对比表
| 对比维度 | OpenAI Whisper | Vosk | FunASR |
| :--- | :--- | :--- | :--- |
| 开源协议 | MIT | Apache 2.0 | Apache 2.0 |
| 语言/方言支持 | 支持99种语言，含多地方言 | 支持20+种语言，方言支持较少 | 以中文为主，方言支持强（含粤语、四川话等） |
| 模型体量 | base模型约1GB，large模型约6GB | 轻量模型几十MB | 模型大小可选，轻量版约100MB |
| 推理速度 | base模型在CPU上可实时运行 | 极快，低延迟 | 轻量版可实时，大模型需GPU加速 |
| 部署难度 | 极低，一行命令安装 | 低，支持多语言SDK | 中等，依赖较多组件 |
| 流式/实时支持 | 非原生支持，需二次开发 | 原生支持流式识别 | 原生支持流式识别 |
| 依赖与部署 | 仅需Python+PyTorch | 无 heavy 依赖，可离线运行 | 依赖ONNX Runtime，部署稍复杂 |

## 3. 选型理由
本次实验选择 **OpenAI Whisper**，理由如下：
1.  安装部署最简单，无需复杂配置，新手友好；
2.  对中文识别准确率高，自带标点断句，输出文本可读性强；
3.  base模型体量小，普通笔记本的CPU就能跑，不需要GPU；
4.  支持直接读取本地音频文件，完美匹配任务二导出的配音音频。
